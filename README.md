# Parks 로컬 다이빙 랜딩 페이지

세부, 보홀, 코타키나발루, 발리에 위치한 PADI 5 Star 공식 다이빙 샵 "Parks 로컬 다이빙"의 고성능 랜딩 페이지입니다.

## 🌊 프로젝트 개요

비싸기만 했던 스쿠버 다이빙을 저렴하고 안전하게 제공하는 Parks 로컬 다이빙의 브랜드 가치를 극대화하는 랜딩 페이지입니다.

## 🛠️ 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 기반 CSS 프레임워크
- **Vite** - 빠른 개발 환경

## 📱 주요 기능

### 1. Hero Section
- 강력한 가치 제안 (Value Proposition)
- "#Parks로컬 다이빙" 브랜드 강조
- 세부/보홀 다이빙 성지 소개

### 2. Social Proof
- 1,000명 이상의 실제 고객 리뷰
- 5점 만점 별점 시스템
- 생생한 고객 후기

### 3. Why Us Section
- 올 인클루시브 다이빙 (추가금 없음)
- 1회 최소 35분 이상 다이빙 시간 보장
- 전세계 4개 지점 운영
- PADI 5 Star 공식 인증

### 4. 지점 소개
- 세부 (Cebu) 1호점
- 보홀 (Bohol) 2호점
- 코타키나발루 3호점
- 발리 (Bali) 4호점

### 5. 일정 & 포인트
- 상세 다이빙 투어 일정
- 각 지점별 다이빙 포인트 소개
- 타임라인 시각화

### 6. 가격 비교
- 타업체 대비 투명한 가격 비교
- 최대 30% 저렴한 가격
- 최대 7만원 절약 강조

### 7. 이벤트 & CTA
- 네이버 포인트 리뷰 이벤트
- 고프로 영상 무료 공유
- 연인/친구와 함께 다이빙 가능

## 🚀 시작하기

### 설치

\`\`\`bash
npm install
\`\`\`

### 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

브라우저에서 `http://localhost:5173` 열기

### 프로덕션 빌드

\`\`\`bash
npm run build
\`\`\`

### 프리뷰

\`\`\`bash
npm run preview
\`\`\`
